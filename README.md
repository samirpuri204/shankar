# श्री शंकर गिरी - व्यक्तिगत पोर्टफोलियो वेबसाइट

यो श्री शंकर गिरीको व्यक्तिगत पोर्टफोलियो वेबसाइट हो। उहाँ कुश्मा, पर्बतका एक प्रतिष्ठित व्यक्तित्व हुन् र प्रदेश अस्पताल कुश्मा पर्बतको पूर्व अध्यक्ष हुनुहुन्छ।

## विशेषताहरू

- **पूर्ण नेपाली भाषामा**: सम्पूर्ण वेबसाइट नेपाली भाषामा डिजाइन गरिएको
- **मोबाइल फ्रेन्डली**: सबै डिभाइसहरूमा राम्रोसँग काम गर्छ
- **व्यापक सामग्री**: ८ मुख्य सेक्शनहरू सहित
- **इन्टरएक्टिभ डिजाइन**: एनिमेसन र काउन्टर इफेक्टहरू
- **फोटो ग्यालेरी**: २००+ फोटोहरूको संग्रह
- **भिडियो ग्यालेरी**: अन्तर्वार्ता र कार्यक्रमका भिडियोहरू
- **उपलब्धिहरू**: ५०+ कामहरू र योगदानहरूको विस्तृत सूची
- **सामाजिक लिंकहरू**: फेसबुक र टिकटक प्रोफाइलहरू

## वर्तमान पदहरू

- नेपाल रेडक्रस सोसाइटी पर्बत जिल्ला शाखा सल्लाहकार
- श्री शिवालय माध्यमिक विद्यालय व्यवस्थापन समिति सल्लाहकार
- पर्बत र्याण्डम रिडर्स सोसाइटी शाखा सचिव

## फाइल संरचना

```
shankar-giri-portfolio/
├── index.html          # मुख्य HTML फाइल
├── styles.css          # CSS स्टाइलहरू
├── script.js           # JavaScript कार्यक्षमता
├── images/             # फोटो र भिडियोहरू
│   ├── *.JPG          # फोटो फाइलहरू
│   ├── *.MP4          # भिडियो फाइलहरू
│   └── *.MOV          # भिडियो फाइलहरू
└── README.md          # यो फाइल
```

## कसरी चलाउने

1. `index.html` फाइललाई कुनै वेब ब्राउजरमा खोल्नुहोस्
2. वा स्थानीय सर्भर चलाउनुहोस्:
   ```bash
   # Python 3 प्रयोग गरेर
   python -m http.server 8000
   
   # Node.js को http-server प्रयोग गरेर
   npx http-server
   ```
